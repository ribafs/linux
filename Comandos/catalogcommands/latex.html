latex

latex arquivo
  • O argumento arquivo corresponde a um texto ASCII com macros TEX.

Descrição

  • Este comando gera arquivo .dvi a partir da definição de macros TEX.
  • O arquivo gerado com este comando tem extensão .dvi e pode ser formatado para postscrip (extensão ps) com o aplicativo dvips.

Exemplos

Suponha a existêcia do arquivo teste.tex com as macros TEX. Pode-se usar o comando

latex teste
ou
latex teste.tex
para gerar o arquivo teste.dvi.

Observações

  • A extensão tex não é obrigatória nos arquivos compilados pelo comando latex.
  • O arquivo pode ter qualquer extensão, mas neste caso ela deve ser fornecida no comando.
  • Caso o nome do arquivo seja fornecido sem uma extesão, o latex procura pelo nome fornecido. Não o encontrando, procura o arquivo com a extensão tex.

Tópicos relacionados

  • dvips - converte arquivo .dvi em .ps (PostScript).
  • gv - aplicativo para visualização de arquivos com extensão ps e pdf.
  • ps2pdf - converte arquivo no formato ps para o formato pdf.
  • tex- sistema de processamento de textos para formatação de documentos.
