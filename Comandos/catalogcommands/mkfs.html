mkfs

mkfs [opções] dispositivo [blocos]
  • O argumento dispositivo é o arquivo especial correspondente ao dispositivo.
  • O argumento blocos é quantidade de blocos a ser utilizada pelo sistema de arquivos.

Descrição

Este comando constrói um sistema de arquivos Linux.

Algumas opções do comando

  • -t tipo : especifica o tipo de sistema de arquivos a ser criado como, por exemplo, ext2, ext3 e ext4.
  • --help : lista as opções disponíveis com o comando.
  • --version : exibe informação sobre o comando.

Exemplo

O comando

sudo mkfs -t ext3 /dev/hda2
ou
sudo mkfs.ext3 /dev/hda2
cria um sistema de arquivos ext3 na segunda partição.
