glibc

Descrição

  • Glib siginifica GNU C Library e é a implementação da biblioteca padão C do Projeto GNU.
  • É a biblioteca C usada na maioria das distribuições Linux para definir as chamadas de sistema (system calls) e as funções básicas como fopen(), malloc(), printf(), exit(), etc.
  • Esta biblioteca é compatível com padões como ISO (International Standardization Organization) e POSIX (Portable Operating System Interface for Computer Environments).

Observações

Para mais informações, veja o site oficial da glibc em www.gnu.org/software/libc.
