unalias

unalias [apelido]

Descrição

Comando interno do shell que remove uma definição feita pelo comando alias.

Exemplo

Suponha a existência de um alias com o nome teste. Para removê-lo, basta digitar

unalias teste

Observações

  • Para obter mais informações sobre alias, digite
    help unalias
