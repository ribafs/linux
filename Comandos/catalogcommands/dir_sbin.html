/sbin

Descrição

Este diretório contém os aplicativos usados na administração do sistema.

Exemplos de arquivos

  • fsck - Este arquivo é usado para verificar e, opcionalmente, reparar um sistema de arquivos do Linux. Na realidade, o fsck é simplesmente um intermediário para os vários verificadores de sistemas de arquivos disponíveis no Linux. Por exemplo, fsck.ext2 é chamado para verificar sistemas de arquivo do tipo ext2.
  • hwclock - exibe ou modifica a data e a hora do relógio da máquina.
  • init - o processo init é o primeiro processo iniciado pelo sistema e é o processo pai de todos os outros processos.
  • insmod - carrega módulos do kernel na memória do sistema.
  • lsmod - lista os módulos do kernel que estão carregados na memória.
  • modprobe - carrega e remove módulos do kernel.
  • rmmod - descarrega módulos do kernel da memória do sistema.
