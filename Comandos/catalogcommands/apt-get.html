apt-get

apt-get [opções]

Descrição

Este utilitário permite a instalação, atualização e remoção de pacotes do sistema.

Algumas opções do comando

  • -h, --help : fornece as opções do utilitário.
  • -u, --show-upgraded : mostra a lista de pacotes sendo atualizada.
  • install pacotes : atualiza os pacotes especificados.
  • remove pacotes : remove os pacotes especificados.
  • update : atualiza a lista das versões dos pacotes disponíveis.
  • upgrade : atualiza os pacotes instalados no sistema.

Exemplos

  • Para atualizar a lista das versões dos pacotes disponíveis nos repositórios listados em /etc/apt/sources.list, digite
    sudo apt-get update
  • Para atualizar os pacotes instalados no sistema, basta digitar
    sudo apt-get -u upgrade
    Naturalmente, é importante que o comando apt-get seja executado primeiro com a opção update. Isto garantirá que o sistema será atualizado com as versões mais novas disponíveis.
  • Para atualizar uma distribuição inteira sincronizando os pacotes (já instalados) para que sejam compatíveis entre si.
    apt-get -u dist-upgrade
  • Para instalar o analisador de tráfego de redes wireshark, digite
    sudo apt-get install wireshark
    O utilitário então avalia o que já está instalado e o que ainda precisa ser instalado. Abaixo, uma possível resposta ao comando.
    Lendo listas de pacotes... Pronto
    Construindo árvore de dependências
    Lendo informação de estado... Pronto
    Os seguintes pacotes foram instalados automaticamente e já não são necessários:
       linux-headers-3.2.0-29 linux-headers-3.2.0-29-generic-pae
    Use 'apt-get autoremove' para removê-los.
    Os pacotes extra a seguir serão instalados:
       libc-ares2 libsmi2ldbl libwireshark-data libwireshark1 libwiretap1 libwsutil1 wireshark-common
    Pacotes sugeridos:
       snmp-mibs-downloader wireshark-doc
    Os NOVOS pacotes a seguir serão instalados:
       libc-ares2 libsmi2ldbl libwireshark-data libwireshark1 libwiretap1 libwsutil1 wireshark wireshark-common
    0 pacotes atualizados, 8 pacotes novos instalados, 0 a serem removidos e 0 não atualizados.
    É preciso baixar 12,8 MB de arquivos.
    Depois desta operação, 49,0 MB adicionais de espaço em disco serão usados.
    Você quer continuar [S/n]?
    Se o usuário responder "s", a instalação será completada.
  • Para remover a ferramenta wireshark, digite
    sudo apt-get remove wireshark
    O utilitário verifica os pacotes a serem removidos junto com o wireshark e solicita confirmação para o usuário.
    sudo apt-get remove wireshark
    Lendo listas de pacotes... Pronto
    Construindo árvore de dependências
    Lendo informação de estado... Pronto
    Os seguintes pacotes foram instalados automaticamente e já não são necessários:
       linux-headers-3.2.0-29 libwsutil1 linux-headers-3.2.0-29-generic-pae libwireshark-data libsmi2ldbl libc-ares2 libwireshark1
       libwiretap1 wireshark-common
    Use 'apt-get autoremove' para removê-los.
    Os pacotes a seguir serão REMOVIDOS:
       wireshark
    0 pacotes atualizados, 0 pacotes novos instalados, 1 a serem removidos e 0 não atualizados.
    Depois desta operação, 2.189 kB de espaço em disco serão liberados.
    Você quer continuar [S/n]?
    Se o usuário responder "s", a ferramenta wireshark será removida.
