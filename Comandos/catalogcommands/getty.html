getty

Descrição

O getty é o segundo dos três programas (init, getty e login) usados pelo sistema para permitir o acesso dos usuários. Ele é iniciado pelo processo init para:

  1. Abrir as linhas tty e configurar o seu modo de funcionamento.
  2. Imprimir a identificação de acesso ao sistema e receber o nome do usuário.
  3. Iniciar o processo login para o usuário.

Na realidade, o getty é um programa que gerencia um terminal. Seu objetivo é proteger o sistema de acesso não autorizado. Os terminais gerenciados pelos processos getty são implementados como consoles virtuais. Por padrão, o DuZeru inicializa seis consoles virtuais com o processo getty que podem ser acessadas usando as teclas de Ctrl+Alt+F1 a Ctrl+Alt+F6, enquanto as teclas CTRL+ALT+F7 estão associadas à interface gráfica.

Observações

  • O nome getty significa "get teletype".
  • O arquivo /etc/issue é usado pelo getty para fornecer informações no login local, enquanto o arquivo /etc/issue.net fornece informações no login remoto.
  • O DuZeru também disponibiliza o programa "agetty" (alternative getty) para inicializar os terminais virtuais. Esta versão do getty tem a vantagem de não usar arquivos de configuração, pois os parâmetros são fornecidos na linha de comando.
