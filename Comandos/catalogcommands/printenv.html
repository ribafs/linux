printenv

printenv [opções] [variáveis]
  • O argumento variáveis corresponde aos nomes das variáveis de ambiente.

Descrição

Este comando exibe informações sobre as variáveis de ambiente.

Algumas opções do comando

  • --help : exibe as opções do comando.
  • --version : exibe informações sobre o comando.

Exemplo

O comando

printenv USER SHELL
mostra o login e o shell utilizados pelo usuário.

Observações

  • O uso do comando printenv sozinho, sem o nome de qualquer variável de ambiente, faz com que os valores de todas as variáveis de ambiente sejam exibidos.
