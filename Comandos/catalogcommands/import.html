import

import [opções]

Descrição

Produz um arquivo de imagem a partir da janela visível de um servidor X.

Algumas opções do comando

  • compress type : tipo da compressão do pixel ao produzir uma imagem.
  • quality value : nível de compressão JPEG/MIFF/PNG.
  • resize geometry : redimensiona a imagem.
  • window id : seleciona janela com o id ou nome especificado.

Exemplos

  • Para selecionar uma janela X ou uma área da tela com o mouse e salvar em formato .eps, digite
    import figure.eps
  • Para caputurar a tela inteira do servidor X no formato jpeg, sem o mouse e com o nome "tela.jpeg", basta digitar
    import -window root tela.jpeg

Observações

  • O formato padrão das imagens é o MIFF (Magick Image File Format).
  • O comando import faz parte das ferramentas do software para tratamento de imagens chamado ImageMagick.
