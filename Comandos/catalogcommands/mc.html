mc

mc [opções]

Descrição

O mc (midnight commander) é um browser de diretórios e gerenciador de arquivos de sistemas operacionais da família Unix.

São algumas das opções deste comando

  • -a : desabilita o uso de caracteres gráficos.
  • -c : usa cores.
  • -d : desabilita o mouse.

São exemplos de teclas que você pode usar no aplicativo

  • F1 : ajuda.
  • F2 : exibe menu do usuário.
  • F3 : visualiza o conteúdo do arquivo.
  • F4 : edita arquivo.
  • F5 : copia arquivos.
  • F6 : move arquivos.
  • F7 : cria diretório.
  • F8 : remove arquivos.
  • F9 : move cursor para o menu superior do mc.
  • F10 : encerra mc.
