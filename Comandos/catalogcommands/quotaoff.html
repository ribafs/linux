quotaoff

quotaoff [opções] [sistema de arquivos]
  • O argumento sistema de arquivos é o nome do sistema de arquivos cujo sistema de cotas deve ser desabilitado.

Descrição

Este comando desabilita o sistema de cotas de um sistema de arquivos.

Algumas opções do comando

  • -a : desabilita as cotas de todos os sistemas de arquivos definidos no /etc/fstab.
  • -u : desabilita as cotas de usuários. É o padrão.
  • -g : desabilita as cotas dos grupos.

Exemplo

O comando

quotaoff /dev/hda2
desabilita o sistema de cotas do /dev/hda2.

Observações

  • É preciso usar o comando sudo para obter informações sobre o sistema de cotas de outros usuários.

Tópicos relacionados

  • cotas de disco - define a quantidade de espaço em disco que cada usuário/grupo pode usar.
  • quota - fornece informações sobre o sistema de cotas do disco.
  • quotaon - habilita o sistema de cotas de um sistema de arquivos.
