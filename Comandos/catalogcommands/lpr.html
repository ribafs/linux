lpr

lpr [-P  impressora] arquivo
  • O argumento impressora é o nome de uma impressora no sistema.
  • O argumento arquivo é o nome do arquivo a ser impresso.

Descrição

Este comando imprime arquivo.

Observações

  • Se o nome da impressora não for fornecido, será usada a impressora padrão do sistema.

Tópicos relacionados

  • lpc - aplicativo para controle de impressão de arquivos.
  • lpq - exibe o status da fila de impressão.
  • lprm - remove arquivos da fila de impressão.
