MBR

Descrição

  • O MBR (Master Boot Record) possui informações sobre como inicializar o sistema e sobre as partições do HD.
  • O MBR é criado quando um HD é particionado, mas ele não faz parte de qualquer partição.
  • Normalmente, o MBR é armazenado no primeiro setor do HD, ou seja, é localizado no cilindro 0, cabeça (head) 0 e setor 1.
  • O MBR contém as seguintes estruturas:
    • Master Boot Code - programa inicial carregado pelo BIOS para inicializar o processo de boot do sistema. Este programa transfere o controle para o programa que vai carregar o sistema operacional.
    • Master Partition Table - contém informações sobre as partições.

Inicialização do LINUX

Quando a máquina é ligada, o BIOS (Basic Input/Output System) assume o controle:

  • ler informações sobre o hardware da máquina;
  • executa o POST (Power On Self Test) que inicializa e testa o hardware da máquina;
  • procura pelo carregador do sistema operacional (ex: lilo, grub, etc) no MBR ou na primeira partição do dispositivo de armazenamento (ex: HD, CD, DVD, pendriver, etc);
  • o carregador do sistema passa o controle para o Linux Kernel;
  • o kernel dá início ao processo init que carrega todos os serviços Linux.

Observações

  • O BIOS é normalmente armazenado na memória ROM (Read-only memory) da placa-mãe.

Tópicos relacionados

  • grub - carregador de inicialização do Linux.
  • lilo - carregador de inicialização do Linux.
