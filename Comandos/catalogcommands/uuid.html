UUID

Descrição

UUID (Universally Unique Identifier) é um campo de 128 bits usado no DuZeru para identificar as partições do disco.

Exemplo

Para ver as UUIDs das partições do disco, digite

ls -l /dev/disk/by-uuid/
No exemplo abaixo são apresentadas as UUIDs de quatro partições de uma máquina.
total 0
lrwxrwxrwx 1 root root 10 Fev 13 08:26 12a14f23-93fc-474e-97b3-6e0773a150e0 -> ../../sda5
lrwxrwxrwx 1 root root 10 Fev 13 08:26 64B4AD31B4AD0720 -> ../../sda2
lrwxrwxrwx 1 root root 10 Fev 13 08:26 7CC094C0C09481D6 -> ../../sda1
lrwxrwxrwx 1 root root 10 Fev 13 08:26 b2caf857-2b09-46cb-934d-2fb411f7fa6b -> ../../sda6
Entretanto, o arquivo /etc/fstab do mesmo sistema só tem 2 partições.
proc /proc proc nodev,noexec,nosuid 0 0
# / was on /dev/sda6 during installation
UUID=b2caf857-2b09-46cb-934d-2fb411f7fa6b / ext4 errors=remount-ro 0 1
# swap was on /dev/sda5 during installation
UUID=12a14f23-93fc-474e-97b3-6e0773a150e0 none swap sw 0 0
Esta diferença ocorre porque o arquivo /etc/fstab só mostra o que está montado no sistema (neste caso, temos a partição sda6 com o sistema Linux e a partição sda5 como área de swap), enquanto o primeiro comando apresenta todas as partições detectadas (as partições sda1 e sda2 são Windows e não estão montadas).

Para ver como as partições estão alocadas, basta digitar

sudo fdisk -l
Abaixo, é apresentada a saída do comando para o mesmo ambiente discutido anteriormente.

Também é possível ver as UUIDs no /boot/grub/grub.cfg que é o arquivo de configuração do GRUB.

Tópicos relacionados

  • fdisk - aplicativo utilizado para particionar discos.
  • GRUB - carregador de inicialização do Linux.
  • mount - monta sistemas de arquivos.
