ipcrm

ipcrm [opções] id
  • O argumento id identifica o recurso a ser removido.

Descrição

Este comando remove um objeto ipc (interprocess communication) que pode ser: memória compartilhada, filas de mensagens e semáforos.

Algumas opções do comando

  • -M shmkey : remove o segmento de memória compartilhada criado com shmkey.
  • -m shmid : remove o segmento de memória compartilhada identificado por shmid.
  • -Q msgkey : remove a fila de mensagens criada com msgkey.
  • -q msgid : remove a fila de mensagens identificada por msgid.
  • -S semkey : remove semáforo criado com semkey.
  • -s semid : remove semáforo identificado por semid.

Tópicos relacionados

  • ipcs - fornece informações sobre recursos ipc.
