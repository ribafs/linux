telnet

telnet [opções] host
  • O argumento host é o nome DNS (ou endereço IP) da máquina remota.

Descrição

Este utilitário é uma interface para o protocolo TELNET tornando possível acessar uma máquina remotamente.

Algumas opções do comando

  • -a : tenta login automático.
  • -l usuário : faz a conexão usando usuário como identificação.

Exemplos

O comando

telnet -l aluno 10.1.2.3
solicita uma conexão como o usuário aluno na máquina 10.1.2.3.
