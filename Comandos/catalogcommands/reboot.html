reboot

reboot [opções]

Descrição

Este comando reinicializa o sistema.

Algumas opções do comando

  • -f, --force : força a reinicialização do sistema.
  • --verbose : exibe mensagens mais detalhadas sobe a reinicialzação do sistema.

Observações

  • Para reinicializar o sistema também é possivel usar o comando init ou o comando shutdown.
