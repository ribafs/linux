userdel

userdel [opções] usuário

Descrição

Este comando remove usuário e seus respectivos arquivos do sistema.

Algumas opções do comando

  • -h, --help : exibe as opções do comando.
  • -r, --remove : deleta o diretório home e todos os seus arquivos.

Exemplo

Para deletar o usuário aluno, basta digitar

userdel aluno

Observações

  • O comando userdel deleta todas as entradas relacionadas ao usuário especificado nos seguintes arquivos: /etc/passwd, /etc/shadow e /etc/group.
  • O comando deluser também deleta usuários do sistema.
  • Os comandos adduser e useradd criam usuários.
