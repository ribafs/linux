source

source arquivo

Descrição

Este comando atualiza arquivos do sistema.

Exemplo

Por exemplo, após modificar o seu arquivo de configuração do sistema, você pode usar o comando source para tornar as alterações válidas, sem precisar sair e entrar novamente no sistema.

Para os usuários que usam o shell bash, este arquivo é o .bashrc que fica no diretório inicial do usuário e, portanto, o seguinte comando deve ser digitado

source .bashrc
