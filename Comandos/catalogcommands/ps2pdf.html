ps2pdf

ps2pdf arquivo-de-entrada [arquivo-de-saída]
  • O argumento arquivo-de-entrada é um arquivo no formato ps (PostScript).
  • O argumento arquivo-de-saída é o nome do arquivo que será gerado no formato pdf (Portable Document Format). Se este parâmetro não for fornecido, o aplicativo usa o mesmo nome do arquivo de entrada.

Descrição

Este comando converte um arquivo no formato ps para o formato pdf.

Exemplos

  • Para converter o arquivo "teste.ps" para o formato pdf, criando o arquivo "teste.pdf", basta digitar
    ps2pdf teste.ps
  • Para definir "teste2" como arquivo de saída do comando, entre com
    ps2pdf teste.ps teste2
    Note que, neste caso, o arquivo não terá extensão pdf.

Tópicos relacionados

  • dvips - converte arquivo .dvi (device independent file format) em .ps (PostScript).
  • gv - aplicativo para visualização de arquivos com extensão ps e pdf.
