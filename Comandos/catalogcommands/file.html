file

file arquivo

Descrição

Este comando informa o tipo do arquivo.

Algumas opções do comando

  • --help : mostra informações sobre o aplicativo.
  • --version : mostra a versão instalada do aplicativo.

Exemplos

  • Suponha que o arquivo teste seja um arquivo de texto. O comando
    file teste
    poderá fornecer a seguinte resposta
    teste: UTF-8 Unicode text
  • Para verificar o diretório /etc, digite
    file /etc
    O sistema informa
    /etc: directory

Observações

  • É também possível obter várias informações sobre arquivos usando o comando stat.
