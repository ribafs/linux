mke2fs

mke2fs [opções] dispositivo
  • O argumento dispositivo é o arquivo especial correspondente ao dispositivo.

Descrição

Este comando cria um sistema de arquivos ext2/ext3/ext4.

São algumas das opções deste comando

  • -b num : especifica num bytes por bloco.
  • -c : verifica o dispositivo antes de formatá-lo.
  • -m num : reserva num% de espaço no disco para o administrador.
  • -t tipo : especifica o tipo de sistemas de arquivos como, por exemplo, ext2, ext3 e ext4.

Exemplo

O comando

mke2fs -t ext4 /dev/hda1
formata o dispositivo /dev/hda1 para suportar um sistema de arquivos do tipo ext4.

Observações

  • O arquivo de configuração do mke2fs é o /etc/mke2fs.conf.
