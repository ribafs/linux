useradd

useradd [opções] usuário

Descrição

Este comando cria um novo usuário sem alocar recursos.

Algumas opções do comando

  • -d, --home HOME_DIR : define o nome do diretório home do usuário (mas não o cria).
  • -s, --shell SHELL : define o shell padrão do usuário.
  • -h, --help : exibe as opções do comando.

Exemplos

O comando

sudo useradd -d /home/aluno -s /bin/bash aluno
cria o usuário aluno e define o nome do diretório home (não cria o diretório) e o shell padrão.

Observações

  • O comando useradd apenas adiciona um novo usuário ao arquivo /etc/passwd, ele não aloca qualquer recurso. Use o comando adduser caso queira especificar recursos na alocação como, por exemplo, criação de um diretório home.
  • A configuração padrão usada pelo comando useradd pode ser encontrada em /etc/login.defs.
  • Os comandos addgroup e groupadd criam um novo grupo de usuários no sistema.
  • O comando deluser deleta um usuário ou um grupo do sistema.
  • O comando userdel remove usuário e seus respectivos arquivos do sistema.
