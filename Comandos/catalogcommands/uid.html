UID

Descrição

A sigla UID significa User IDentification. O Linux utiliza o número de UID para monitorar os usuários e para verificar as permissões desses usuários.

Observações

  • O número de UID zero é especial sob o Linux pois é o número do usuário root. Este número tem acesso irrestrito ao sistema Linux.

Tópicos relacionados

  • adduser - cria um novo usuário.
  • GID - número de identificação de grupo para o kernel do Linux.
  • permissão de acesso - autorização para ler/gravar/executar um arquivo/diretório do sistema.
  • root - administrador do sistema ou superusuário.
