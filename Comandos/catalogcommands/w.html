w

w [opções] [usuário]
    O argumento usuário é o nome do usuário do qual se deseja informação.

Descrição

Este utilitário informa quais os usuários que estão conectados e o que eles estão executando.

Algumas opções do comando

  • -f : omite a coluna “FROM” (DE).
  • -h : não mostra o cabeçalho das colunas.
  • -s : listagem curta.
  • -u : não mostra o UID dos processos.
  • -V : mostra informações sobre o utilitário.

Exemplos

O comando

w -s
exibe uma listagem curta dos usuários do sistema e o que estão fazendo.

Observações

  • O comando uptime informa há quanto tempo o sistema está funcionando e o número de usuários conectados.
  • O comando users mostra os usuários que estão atualmente conectados ao sistema.
