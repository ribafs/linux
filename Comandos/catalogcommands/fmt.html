fmt

fmt [opções][arquivo]

Descrição

Este comando formata textos.

Algumas opções do comando

  • -c : preserva a identação das duas primeiras linhas do arquivo.
  • -p : formata apenas linhas que come√ßam com uma determinada string.
  • -u : deixa os espaçamentos entre as palavras uniforme. Um espaço entre palavras e dois entre sentenças.
  • -w : especifica um tamanho máximo de caracteres em uma linha. O padrão é 75.

Exemplos

  • O comando
    fmt -w 40 teste
    formata o arquivo teste com largura máxima de 40 caracteres.
  • Para obter espacejamento uniforme para o arquivo teste, digite
    fmt -u teste

Observações

  • Se o nome do arquivo não é fornecido na linha de comandos ou se o arquivo é igual a -, então fmt ler a entrada padrão.
