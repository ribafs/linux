groupadd

groupadd [opções] grupo

Descrição

Este comando cria um novo grupo de usuários.

Algumas opções do comando

  • -g gid : fornece a identificação numérica do grupo (GID) sendo criado. O valor padrão é o menor valor, maior que 999, que ainda não está alocado a outro grupo. Valores entre 0 e 999 são tipicamente reservados para as contas do sistema.
  • -f : força a criação do grupo. Caso o GID especificado já exista, o sistema escolhe outro valor (usado em conjunto com a opção -g).
  • -r : cria uma conta do sistema. Por padrão, o valor escolhido é o menor valor disponível entre 0 e 999, a menos que a opção -g seja usada.

Exemplo

O comando

groupadd -r teste
cria um grupo do sistema chamado teste.

Observações

  • A configuração padrão usada pelo comando groupadd pode ser encontrada em /etc/login.defs.
  • A lista dos grupos existentes no sistema é armazenada em /etc/group.
  • O comando useradd nomalmente exige o uso do comando sudo.
  • O comando addgroup também cria grupo de usuários.
  • Os comandos delgroup e groupdel deletam grupos de usuários.
