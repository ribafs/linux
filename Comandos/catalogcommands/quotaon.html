quotaon

quotaon [opções] [sistema de arquivos]
  • O argumento sistema de arquivos é o nome do sistema de arquivos cujo sistema de cotas deve ser habilitado.

Descrição

Este comando habilita o sistema de cotas de um sistema de arquivos.

Algumas opções do comando

  • -a : habilita as cotas de todos os sistemas de arquivos definidos no /etc/fstab.
  • -u : habilita as cotas de usuários. É o padrão.
  • -g : habilita as cotas dos grupos.

Exemplo

Por exemplo, o comando

quotaon /dev/hda2
habilita o sistema de cotas do /dev/hda2 conforme definido em /etc/fstab.

Observações

  • É preciso usar o comando sudo para obter informações sobre o sistema de cotas de outros usuários.

Tópicos relacionados

  • cotas de disco - define a quantidade de espaço em disco que cada usuário/grupo pode usar.
  • quota - fornece informações sobre o sistema de cotas do disco.
  • quotaoff - desabilita o sistema de cotas de um sistema de arquivos.
