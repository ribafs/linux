export

export variável=valor

Descrição

Comando do shell que permite configurar e compartilhar variáveis de ambiente entre diversos programas e bibliotecas acessados a partir do mesmo terminal.

Exemplo

O comando

export TESTE=2

cria a variável de ambiente TESTE e atribui o valor 2 a esta variável.

Observações

  • É importante observar que a variável criada só existe para a sessão (terminal) onde o comando foi digitado.
  • Para garantir que a configuração de uma variável exista em todas as sessões criadas pelo usuário, o comando deve ser incluído no arquivo .bashrc do seu diretório home. Para disponibilizar atualização imediatamente, digite
    source .bashrc
  • O comando unset ou o comando env -u pode ser usado para apagar uma variável de ambiente.
