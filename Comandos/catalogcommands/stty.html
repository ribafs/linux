stty

stty [opções]

Descrição

Este comando modifica e/ou mostra as configurações de linhas de terminal.

Algumas opções do comando

  • -a : exibe todas as configurações atuais do terminal.
  • -g : exibe todas as configurações atuais do terminal em um formato que pode ser usado como argumento para outro comando stty para restaurar as configurações atuais.
  • --help : informa as opções do aplicativo.
  • --version : exibe informações sobre o aplicativo.
  • sane : restaura a configuração padrão no terminal.

Exemplos

  • O comando
    stty
    exibe informações sobre a comunicação do terminal. Um exemplo é mostrado abaixo.
    speed 38400 baud; line = 0;
    eol = M-^?; eol2 = M-^?; swtch = M-^?;
    ixany iutf8
  • Para ver todas as configurações atuais do terminal, digite
    stty -a
    Abaixo uma possível saída é mostrada. Note, por exemplo, que o terminal tem atualmente 48 linha com 170 colunas cada (o usuário pode modificar o tamanho da tela).
    speed 38400 baud; rows 48; columns 170; line = 0;
    intr = ^C; quit = ^\; erase = ^?; kill = ^U; eof = ^D; eol = M-^?; eol2 = M-^?; swtch = M-^?; start = ^Q; stop = ^S; susp = ^Z; rprnt = ^R; werase = ^W; lnext = ^V;
    flush = ^O; min = 1; time = 0;
    -parenb -parodd cs8 hupcl -cstopb cread -clocal -crtscts
    -ignbrk brkint -ignpar -parmrk -inpck -istrip -inlcr -igncr icrnl ixon -ixoff -iuclc ixany imaxbel iutf8
    opost -olcuc -ocrnl onlcr -onocr -onlret -ofill -ofdel nl0 cr0 tab0 bs0 vt0 ff0
    isig icanon iexten echo echoe echok -echonl -noflsh -xcase -tostop -echoprt echoctl echoke
