pidof

pidof [opções] programa

Descrição

Fornece o PID de um programa em execução.

Algumas opções do comando

  • -s : retorna apenas um PID.
  • -o pid : omite processo com o PID especificado.

Exemplos

  • Para ver quais processos foram iniciados pelo programa init, digite
    pidof init
    Abaixo é mostrada uma possível saída.
    1792 1
  • Para eliminar o processo com PID igual a 1 da lista, basta digitar
    pidof -o 1 init
    A resposta agora é
    1792

Observações

  • O diretório /proc possui a lista dos processos que estão em execução no sistema.
  • O comando ps exibe informações sobre os processos em execução.
  • O comando pstree apresenta, em formato de Ã¡rvore, os processos em execução
  • O comando top lista os processos que estão utilizando a CPU.
