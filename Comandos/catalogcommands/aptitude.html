aptitude

aptitude [opções]

Descrição

  • O aptitude é um front-end para o APT. Este utilitário apresenta um menu que permite a instalação, atualização e remoção de pacotes do sistema.
  • Também é possível usar aptitude na linha de comandos.

Algumas opções

Pode-se clicar nas opções mostradas no cabeçalho do menu ou usar uma das teclas configuradas.

  • "?" : mostra todas as opções do menu (digite "q" para voltar ao menu principal).
  • "+" : adicionar ou atualizar um pacote.
  • "=" : não atualiza o pacote, mesmo que novas versões sejam disponibilizadas.
  • "-" : remove um pacote.
  • "_" : remove um pacote e seus arquivos de configuração.
  • "m" : marca um pacote como "manualmente instalado".
  • "M" : marca um pacote como "automaticamente instalado".
  • "u" : atualiza a lista de pacotes disponíveis.
  • "U" : marca todos os pacotes atualizáveis para serem atualizados.
  • "q" : sai do utilitário.

São algumas opções da linha de comando:

  • install pacote+ : instala o pacote especificado.
  • install pacote+M : instala o pacote e o marca como automaticamente instalado.
  • install pacote- : remove o pacote especificado.
  • install pacote_ : remove o pacote e seus arquivos de configuração.
  • install pacote= : não atualiza o pacote especificado, mesmo que novas versões sejam disponibilizadas.
  • install pacote&M : marca o pacote especificado como automaticamente instalado.
  • install pacote&m : marca o pacote especificado como manualmente instalado.
  • remove pacote : remove o pacote especificado.
  • upgrade : atualiza os pacotes instalados no sistema se tiverem novas versões disponíveis.
  • update : atualiza a lista dos pacotes disponíveis.
  • -h, --help : mostra as opções de comando.
  • --version : exibe informações sobre o utilitário.

Exemplos

  • Para instalar o analisador de tráfego de redes wireshark, digite
    sudo aptitude install wireshark+
    O utilitário então avalia o que já está instalado e o que ainda precisa ser instalado. Abaixo, uma possível resposta ao comando.
    Os NOVOS pacotes a seguir serão instalados:
         wireshark
    Os pacotes a seguir serão REMOVIDOS:
         firefox-locale-de{u} firefox-locale-zh-hans{u} kde-l10n-de{u} kde-l10n-engb{u} kde-l10n-pt{u} kde-l10n-ptbr{u} kde-l10n-zhcn{u}
         language-pack-de{u} language-pack-de-base{u} language-pack-kde-de{u} language-pack-kde-de-base{u} language-pack-kde-en{u}
         language-pack-kde-en-base{u} language-pack-kde-pt{u} language-pack-kde-pt-base{u} language-pack-kde-zh-hans{u}
         language-pack-kde-zh-hans-base{u} language-pack-zh-hans{u} language-pack-zh-hans-base{u} linux-headers-3.2.0-29{u}
         linux-headers-3.2.0-29-generic-pae{u}
    0 pacotes atualizados, 1 novos instalados, 21 a serem removidos e 4 não atualizados.
    É preciso obter 0 B/821 kB de arquivos. Depois do desempacotamento, 233 MB serão liberados.
    Você deseja continuar? [Y/n/?]
  • Para remover a ferramenta wireshark, digite
    sudo aptitude install wireshark-
    ou
    sudo aptitude remove wireshark
  • É possível instalar e remover pacotes em um mesmo comando. Por exemplo,
    sudo aptitude install wireshark+ acpidump-
    pede para instalar a ferramenta wireshark e remover o utilitário acpidump.
