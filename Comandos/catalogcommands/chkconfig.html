chkconfig

chkconfig [opções]

Descrição

Este utilitário habilita ou desabilita serviços do sistema

Algumas opções do comando

  • -t serviço, --terse serviço : informa se o serviço especificado está habilitado ou desabilitado.
  • -l serviço, --list serviço : informa, para cada runlevel, se o serviço fica habilitado ou desabilitado.
  • serviço on : habilita o serviço.
  • serviço off : desabilita o serviço.
  • -a serviço, --add serviço : adiciona o serviço.
  • -d serviço, --del serviço : deleta o serviço.

Exemplos

  • O comando
    chkconfig --list
    lista a configuração de todos os serviços em todos os níveis de execução.
  • Para ver a configuração do serviço cron nos diversos níveis de execução, digite
    chkconfig --list cron
    Suponha que a resposta seja
    cron 0:off 1:off 2:off 3:off 4:off 5:off 6:off
    Para habilitar o cron nos runlevels 2, 3, 4 e 5, basta digitar
    sudo chkconfig cron 2345
    Isto significa que o serviço cron será inicializado sempre que os runlevels de 2 a 5 forem usados. Para verificar a nova configuração, repita o primeiro comando e a resposta será
    cron 0:off 1:off 2:on 3:on 4:on 5:on 6:off
    Para desabilitar o serviço digite
    sudo chkconfig cron off

Observações

  • É possível que o comando chkconfig apresente a seguinte mensagem:
    /sbin/insserv: Arquivo ou diretório não encontrado
    Isto apenas significa que está faltando um link. Para resolver este problema, basta digitar:
    sudo ln -s /usr/lib/insserv/insserv /sbin/insserv
  • Quando o nome do serviço não é informado, o aplicativo assume que o comando deve ser aplicado para todos os serviços.
