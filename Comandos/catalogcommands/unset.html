unset

unset variável

Descrição

Comando do shell que apaga uma variável de ambiente.

Exemplo

O comando

unset TESTE
apaga a variável de ambiente TESTE.

Observações

  • O comando export permite configurar e compartilhar variáveis de ambiente entre diversos programas e bibliotecas acessados a partir de uma mesma sessão.
  • O comando env -u também pode ser usado para apagar uma variável de ambiente.
