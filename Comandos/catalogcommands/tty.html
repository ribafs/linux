tty

tty

Descrição

Este comando mostra o nome do arquivo conectado à entrada padrão.

Algumas opções do comando

  • --help : lista as opções disponíveis com o comando.
  • --version : exibe informação sobre o comando.

Exemplo

  • O comando
    tty
    fornece normalmente a seguite saída
    /dev/pts/0
    Isto significa que o terminal, em que o comando foi digitado, está associado à entrada padrão 0 (na realidade, 0 é um descritor do arquivo). Se o usuário abrir outros terminais, eles terão sequencialmente a numeração 1, 4, 5, 6, ... Os números 2 e 3 não são usados na criação de novos terminais no DuZeru, pois estão associados, respectivamente, ao modem-manager (controla funções de modem) e ao LightDM (gerenciador de janelas X).
  • Suponha que um usuário abriu dois terminais: 0 e 1. No terminal 0, se ele digitar
    ls > /dev/pts/1
    a resposta do comando aparecerá no outro terminal, o terminal 1.

Observações

  • TTY significa "TeleTYpe" que corresponde ao tipo dos primeiros terminais usados com os computadores.
  • Quando um processo abre /dev/ptmx, ele obtém um descritor de arquivo para um PTM (pseudoterminal master) e um descritor de arquivo para PTS (pseudoterminal slave) é criado. Cada descritor de arquivo obtido pela abertura do /dev/ptmx é um PTM independente com o seu próprio PTS.
  • Para obter mais informações sobre o comando tty, digite
    info tty
