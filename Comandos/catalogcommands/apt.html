apt

Descrição

O apt (Advanced Packaging Tool) é um gerenciador de pacotes usado pela distribuição Debian e derivadas, como o DuZeru. Ele permite que o sistema obtenha pacotes para a instalação de aplicativos a partir de repositórios.

Observações

  • O apt é composto por alguns programas, como apt-get e apt-cache, que podem ser usados a partir da linha de comando ou de um front-end como o aptitude. O Synaptic é uma interface gráfica para o apt.
  • Os repositórios utilizados pelo sistema para instalar/atualizar os pacotes estão em /etc/apt/sources.list.
  • Na distribuição DuZeru, o gerenciamento dos pacotes apt é feito pela Central de Programas do DuZeru cujo ícone, por padrão, fica no lado esquerdo da tela.
