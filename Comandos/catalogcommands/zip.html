zip

zip [arquivo-compactado] [arquivo-original]

Descrição

Este utilitário compacta/descompacta arquivos.

Exemplos

  • Para compactar os arquivos do diretório linux criando o arquivo teste.zip, digite
    zip teste linux/*
    A terminação .zip é então adicionada ao nome do arquivo compactado. Portanto, não é necessário definir a extensão do arquivo. Você pode inclusive usar o mesmo nome do arquivo original, pois o arquivo original continua existindo após a compactação.
  • Para descompactar, digite
    unzip teste.zip

Observações

  • São também ferramentas para compactação/descompactação de arquivos: gzip, bzip2 e compress.
  • Para agrupar/desagrupar arquivos em um repositório use tar.
