ypcat

ypcat [opções] map

Descrição

Este comando exibe informações da base de dados do NIS (Network Information System).

Algumas opções do comando

  • -d domínio : especifica domínio diferente do domínio padrão.
  • -h host : especifica informações do host em vez do host padrão.
  • -k : exibe as chaves e seus valores.
  • -t : força o comando ypcat a ignorar a tabela de conversão de apelidos e procura apenas o mapa especificado pelo parâmetro NomeMapa.
  • -x : lista os apelidos conforme especificado pelo parâmetro NomeMapa.

Exemplos

  • Para ver os apelidos especificados para os mapas disponibilizados pelo ypcat, basta digitar
    ypcat -x
    Abaixo é apresentada a resposta para o comando.
    Use "ethers" for map "ethers.byname"
    Use "aliases" for map "mail.aliases"
    Use "services" for map "services.byname"
    Use "protocols" for map "protocols.bynumber"
    Use "hosts" for map "hosts.byname"
    Use "networks" for map "networks.byaddr"
    Use "group" for map "group.byname"
    Use "passwd" for map "passwd.byname"
  • Portanto, para ver as máquinas associadas com o servidor NIS, digite
    ypcat hosts
    Se o servidor estiver funcionando a saída do comando acima devera ser algo do tipo:
    127.0.0.1----localhost----localhost.unixbrasil.com.br
    127.0.0.1----ameixa-------ameixa.unixbrasil.com.br
    127.0.0.1----abacaxi------abacaxi.unixbrasil.com.br
    127.0.0.1----uva----------uva.unixbrasil.com.br
    127.0.0.1----banana-------banana.unixbrasil.com.br

Observações

  • Para usar este comando é preciso instalar o NIS (Network Information Service).
  • O termo YP vem de Yellow Page (páginas amarelas), nome original do NIS.
