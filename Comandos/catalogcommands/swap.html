swap

Descrição

O swap é uma área de troca usada para aumentar a quantidade de memória RAM do sistema.

Observações

  • O tamanho máximo de uma área de swap depende da arquitetura e da versão do kernel.
    • Arquiteturas i386, PPC, m68k e ARM: 2GBytes;
    • Arquitetura mips: 512 MBytes;
    • Arquitetura alpha: 128 GBytes;
    • Arquitetura sparc64: 3 TBytes;
    • kernel com versão superior a 2.3.3: não há limite.
  • O Linux permite a definição de até 32 áreas de swap.
  • O arquivo /proc/swaps possui a configuração da área de swap utilizada pelo sistema.

Tópicos relacionados

  • free - exibe a quantidade de memória livre/usada no sistema.
  • mkswap : formata uma área de swap.
  • swapoff : desabilita dispositivos de swap.
  • swapon : habilita dispositivos de swap.
