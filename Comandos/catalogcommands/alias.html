alias

alias [nome alternativo='comandos']

Descrição

Comando interno do shell que permite definir apelidos (do inglês alias) para comandos.

Exemplo

Por exemplo, o comando

alias teste='ls -al'
define que, toda vez que o usuário digitar "teste", o sistema processará o comando "ls -al".

Observações

  • É importante observar que a definição feita com o comando alias não é permanente, ou seja, da próxima vez que o sistema for acessado, ela não existirá mais. Para criar uma definição permanente, o comando alias deve ser incluído em um dos arquivos carregados pelo shell durante o processo de inicialização.

    Por exemplo, o shell bash ler dois arquivos com as definições: .bashrc e /etc/bashrc. O primeiro arquivo possui as definições feitas pelo próprio usuário, enquanto o segundo arquivo possui as definições do sistema e que podem ser usadas por todos os usuários.

    Para usar uma definição imediatamente após a sua inclusão em .bashrc, sem a necessidade de sair do sistema e entrar novamente, basta digitar

    source .bashrc
  • Para obter mais informações sobre alias, digite
    help alias
  • O comando alias sem parâmetros exibe a lista das definições que estão disponíveis ao usuário.
  • Use o comando unalias para remover uma definição feita pelo comando alias.
