strings

strings [opções] arquivo

Descrição

Este utilitário exibe os strings legíveis com pelo menos 4 caracteres de um arquivo executável.

Algumas opções do comando

  • -a : verifica todo o arquivo.
  • -f : coloca o nome do arquivo de entrada no início de cada linha de saída.
  • -v : mostra a versão do aplicativo.

Exemplo

O comando

strings /bin/bash
exibe os strings (legíveis) do arquivo binário /bin/bash.
