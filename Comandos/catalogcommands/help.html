help

help [comando]

Descrição

Este aplicativo exibe informações sobre os comandos internos do shell.

Exemplos

  • O comando
    help
    apresenta a lista dos comandos internos do shell. Isto significa que, ao receber um comando, o shell primeiro verifica se é um dos seus comandos. Se isto for verdade, o próprio shell se responsabiliza pela execução do comando. Se não for verdade, ele passa o comando para o kernel resolver.
  • Para saber mais sobre o comando cd do shell bash, digite
    help cd
