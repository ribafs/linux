free

free [opções]


Descrição

Este comando mostra a quantidade de memória livre e utilizada, a área de swap no sistema, a memória compartilhada e os buffers utilizados pelo kernel.

Algumas opções do comando

  • -k : as informações são dadas em Kbytes (é o padrão).
  • -m : as informações são dadas em Mbytes.
  • -o : oculta a linha com as informações sobre os buffers utilizados pelo kernel.
  • -t : mostra uma linha contendo a quantidade total de memória do sistema, a quantidade de memória livre e a quantidade de memória em uso.
  • -s num : mostra a quantidade de memória livre usada a cada num segundos.

Exemplos

  • Ao entrar com o comando
    free -t
    uma possível resposta é
    totalusedfreesharedbufferscached
    Mem:205221610864809657360 151080 483544
    -/+ buffers/cache:4518561600360
    Swap:20858840 2085884
    Total:413810010887963049304
    O exemplo acima informa que o sistema tem 4GBytes de memória: 2GBytes de memória RAM e 2GBytes de área de swap.
  • Para mostrar, a cada 5 segundos, a quantidade de memória livre e ocupada do sistema, digite
    free -s 5

Observações

  • Também é possível ver a quantidade de memória física no log do dmesg que fica no diretório /var/log/.
    cat /var/log/dmesg | grep Memory
    A saída abaixo mostra que a máquina possui 2GBytes de memória, sendo que parte dela é reservada para uso do Linux.
    Memory: 2037636k/2087744k available (5825k kernel code, 49656k reserved, 2849k data, 744k init, 1174344k highmem)

Tópicos relacionados

  • dmidecode -t memory - exibe informações sobre a memória.
  • slabtop - exibe informações em tempo real sobre a memória cache.
  • vmstat - exibe estatísticas sobre a memória virtual.
