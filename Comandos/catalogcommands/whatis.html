whatis

whatis comandos

Descrição

Este utilitário mostra um resumo sobre um ou mais comandos do Linux.

Algumas opções do comando

  • -h, --help : exibe as opções do utilitário.
  • -V, --version : mostra informações sobre o utilitário.

Exemplo

O comando

whatis man ls
apresenta a saída abaixo.
man (7) - macros to format man pages
man (1) - an interface to the on-line reference manuals
ls (1) - list directory contents

Observações

  • O comando whatis utiliza o índice da base de dados criado/atualizado pelo mandb e, por isso, a pesquisa é feita apenas nos nomes das páginas (comandos, funções, utilitários e aplicativos).
  • O resultado da pesquisa corresponde à descrição que cada página do manual possui.
  • Os comandos apropos e man -k pesquisam por determina expressão nas páginas do manual do sistema.
  • O comando mandb cria/atualiza os índices da base de dados das páginas do manual do sistema.
  • O comando xman é um navegador para as páginas on-line.
