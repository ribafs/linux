ispell

ispell [opções] arquivo...

Descrição

Este comando faz a verificação ortográfica de um texto.

Algumas opções do comando

  • -t : o arquivo de entrada está no formato TEX ou LATEX.
  • -H : o arquivo de entrada está no formato SGML/HTML.
  • -o : o arquivo de entrada deve ser tratado como um arquivo texto comum.
  • -b : cria um arquivo de backup com extensão .bak.
  • -x : deleta o arquivo de backup após a verificação ortográfica.
  • -d : especifica um outro arquivo para ser usado como dicionário pela ferramenta.

Observações

  • Por padrão, o dicionário utilizado pelo aplicativo ispell é o inglês. Mas o Linux fornece dicionários para correção ortográfica de vários idiomas. Para ver qual o dicionário padrão utilizado pelo aplicativo ispell, basta digitar
    printenv DICTIONARY
    Se a variável de ambiente DICTIONARY não apresentar nenhum valor, então o inglês é o padrão.
  • O nome do dicionário de português brasileiro para o ispell é ibrazilian. É possível baixá-lo a partir da Central de Programas do DuZeru. Por exemplo, para usar este dicionário para corrigir o arquivo teste.tex, digite
    ispell -d brazilian -t teste.tex
    Outra forma é alterar a variável de ambiente DICTIONARY
    export DICTIONARY=brazilian
    e, em seguida, usar o ispell
    ispell -t teste.tex
    Note que a alteração feita acima na variável DICTIONARY só é válida enquanto o usuário estiver logado na mesma sessão. Para alterar em caráter permanente o idioma padrão do sistema, deve-se incluir o comando no arquivo .bashrc que fica no diretório home do usuário.
  • Os arquivos de dicionários disponíveis no sistema e utilizados pelo ispell são armazenados em /usr/lib/ispell/. Por exemplo, o dicionário padrão corresponde ao arquivo default.hash e o dicionário brasileiro corresponde ao arquivo brazilian.hash.
