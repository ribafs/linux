rfkill [opções] comandos

Descrição

Ferramenta para habilitar e desabilitar dispositivos sem fio.

Opções

  • --version : mortra informações sobre o aplicativo.

Comandos

  • help : mostra informações sobre o aplicativo.
  • event : ouve e exibe os eventos relacionados à ferramenta.
  • list [type] : lista todos os equipamentos sem fio ou apenas os equipamentos de um determinado tipo.
  • block index|type : Desabilita os equipamentos correspondentes ao índice fornecido. O argumento "type" pode ser "all", "wifi", "wlan", "bluetooth", "uwb", "ultrawideband", "wimax", "wwan", "gps", "fm" ou "nfc".
  • unblock index|type : habilita os equipamentos correspondentes ao índice fornecido.

Exemplo

O comando

rfkill list
informa os equipamentos sem fio da máquina.
0: dell-wifi: Wireless LAN
Soft blocked: no
Hard blocked: no
1: phy0: Wireless LAN
Soft blocked: no
Hard blocked: no

Observações

Para desativar o bluetooth na inicialização do linux, basta colocar no arquivo /etc/rc.local as linhas abaixo.

rfkill block bluetooth
exit 0
