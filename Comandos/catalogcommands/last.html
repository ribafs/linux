last

last [opções]

Descrição

Este comando exibe todas as informações referentes a entrada (login) e saída (logout) de usuários do sistema.

Algumas opções do comando

  • -a : exibe o nome da máquina onde foi efetuado o login.
  • -d : exibe o número de IP da máquina remota onde foi efetuado o login.
  • -f arquivo : define o nome do arquivo de onde serão extraídas as informações de login e logout dos usuários. Por padrão, o arquivo lido é o /var/log/wtmp.
  • reboot : exibe um registro de todas as reinicializações desde a criação do arquivo.
  • -w : exibe informações completas dos usuários e domínios.
  • -x : exibe informações sobre parada do sistema e sobre mudança de runlevel.

Exemplos

  • Ao digitar o comando last, o sistema fornecerá uma saída parecida com a mostrada abaixo.
    aluno pts/2 :0 Mon Dec 31 09:37 stilllogged in
    rebootsystem boot3.2.0-35-genericMon Dec 31 09:33 -10:17(00:43)
    aluno pts/1 :0 Sun Dec 30 20:30 -21:24(00:54)
    rebootsystem boot3.2.0-35-genericSun Dec 30 20:23 -21:25(01:01)
  • O comando
    last -d
    informa os endereços IP das conexões não locais.

Observações

  • Caso nenhum argumento seja passado, o comando last exibe todas as informações armazenadas no arquivo /var/log/wtmp de todos os usuários do sistema.
  • O comando lastb funciona da mesma forma do comando last. Entretanto, ele usa, por padrão, o arquivo /var/log/btmp que possui informações sobre as tentativas mal sucedidas de se logar ao sistema.
