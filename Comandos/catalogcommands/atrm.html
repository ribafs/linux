atrm

atrm [opções] tarefa
  • O argumento tarefa corresponde ao número de identificação da tarefa agendada.

Descrição

Este comando remove tarefas agendadas com o comando at.

Opção do comando

  • -V : fornece a versão do comando atrm e informa para quem se deve reportar erro.

Exemplos

Suponha que a seguinte saída seja fornecida com o comando atq ou com o comando "at -l".

14FriNov9 16:00:00 2012 a aluno
15FriNov9 20:00:00 2012 a aluno
Isto significa que o usuário aluno possui duas tarefas agendadas. A primeira tarefa será executadaàs 16 horas do dia 9 de novembro de 2012, enquanto a segunda tarefa será executada às 20 horas do mesmo dia. Note que o número de identificação destas tarefa são 14 e 15, respectivamente. Para remover apenas a primeira tarefa, basta digitar,

atrm 14
Para deletar as duas tarefas acima, digite
atrm 14 15

Observações

  • O comando atrm é um alias para "at -d".
  • As tarefas agendadas ficam armazenadas em /var/spool/cron/atjobs.

Tópicos relacionados

  • at : agenda tarefas.
  • atq : lista as tarefas agendadas com o comando at.
