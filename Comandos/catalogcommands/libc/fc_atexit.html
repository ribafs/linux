atexit( )

Descrição

Esta função chama outra função quando o processo termina normalmente.

Formato

#include<stdlib.h>
int atexit(void (*function)(void));
  • O argumento function corresponde à função a ser chamada quando o processo termina normalmente.

Valor de retorno

A função retorna zero em caso de sucesso e um valor não zero em caso de erro.

Exemplo

1. #include<stdio.h>
2. #include<stdlib.h>
3. void teste_atexit()
4.    {
5.        printf("Programa terminou sem erros\n");
6.    }
7. int main ()
8.    {
9.         atexit(teste_atexit);
10.       printf("Inicio do programa\n");
11.   }

Comentários

  • Linhas de 1 a 2 : é feita a inclusão dos arquivos headers da biblioteca C.
    • stdio.h - cabeçalho com a definição das funções de E/S dos arquivos como, por exemplo, printf( ).
    • stdlib.h - cabeçalho com a definição da função atexit( ).
  • Linhas de 3 a 6 : função chamada por atexit( ).
  • Linha 7 : início da função main( ) cujos comandos vão da linha 9 a linha 10.

Execução

Para compilar, digite

gcc -o atexit atexit.c
Para executar, basta digitar
./atexit
A saída gerada pelo programa é mostrada abaixo.
Inicio do programa
Programa terminou sem erros

Observações

  • Não há necessidade da existência de uma função exit( ) no programa para que a função definida em atexit( ) seja chamada. Basta que o término da execução seja normal.
  • Quando um erro ocorre durante a execução, o sistema coloca o código do erro na variável externa errno.
  • Não é possível passar argumentos para a função chamada com atexit().
  • Se várias funções são registradas com atexit( ), elas são chamadas na ordem inversa em que foram registradas.
  • A função atexit( ) faz parte da biblioteca padão C. Por isso, é preferível o uso dessa função ao uso da função on_exit().
