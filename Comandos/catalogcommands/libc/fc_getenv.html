getenv( )

Descrição

Esta função fornece o valor de uma variável de ambiente.

Formato

#include <stdlib.h>
char *getenv(const char *name);

Valor de retorno

A função getenv( ) retorna um ponteiro para o valor da variável de ambiente ou retorna NULL se a variável não existir.

Exemplo

1. #include<stdio.h>
2. #include<stdlib.h>
3. #include<string.h>
4. #include<errno.h>
5. int main()
6. {
7.     char *dir;
8.     dir = getenv("HOME");
9.     if (dir == NULL)
10.       printf("Erro getenv() %d: %s\n", errno, strerror(errno));
11.    else
12.       printf("Diretorio home = %s\n", dir);
13.}

Comentários

  • Linhas de 1 a 4 : é feita a inclusão dos arquivos headers da biblioteca C.
    • stdio.h - cabeçalho com a definição das funções de E/S dos arquivos como, por exemplo, printf( ).
    • stdlib.h - cabeçalho com a definição da função getenv( ).
    • string.h - cabeçalho com a definição do tipo string e com várias funções para manipulação de strings como, por exemplo, strerror( ) que retorna a mensagem associada a um determinado número de erro.
    • errno.h - cabeçalho com a definição da variável errno que armazena o número do erro ocorrido na execução do programa.
  • Linha 5 : início da função main( ) cujos comandos vão da linha 7 a linha 12.
  • Linha 7 : define a variável que armazenará o nome de um diretório.
  • Linha 8 : recupera o nome do diretório home do usuário.
  • Linhas de 9 a 12 : verifica se a função getenv( ) foi executada com sucesso. Em caso negativo, imprime o erro ocorrido. Em caso positivo, informa qual é o diretório home do usuário.

Execução

Para compilar, digite

gcc -o getenv getenv.c
Para executar, entre com
./getenv
Supondo que o programa seja executado pelo usuário aluno, a resposta é
Diretorio home = /home/aluno

Observações

  • Para ver a lista de variáveis de ambiente usadas no seu sistema, use o comando printenv.
  • Veja shell para mais informações sobre as variáveis de ambiente.
