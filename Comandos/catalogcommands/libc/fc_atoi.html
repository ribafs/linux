atoi( )

Descrição

Esta função converte uma string em um inteiro.

Formato

#include<stdlib.h>
int atoi(const char *nptr);
  • O argumento *nptr é um ponteiro para uma string.

Valor de retorno

O valor convertido.

Exemplo

1. #include <stdio.h>
2. #include <stdlib.h>
3. int main(int argc, char *argv[])
4.    {
5.        int num;
6.        if (argc != 2)
7.           {
8.              printf("Forneca um numero\n");
9.              return 1;
10.          }
11.       else
12.          num = atoi(argv[1]);
13.       printf("Numero recebido = %d\n", num);
14.   }

Para obter o código-fonte do exemplo clique aqui.

Comentários

  • Linhas de 1 a 2 : é feita a inclusão dos arquivos headers da biblioteca C.
    • stdio.h - cabeçalho com a definição das funções de E/S dos arquivos como, por exemplo, printf( ).
    • stdlib.h - cabeçalho com a definição da função atoi( ).
  • Linha 3 : início da função main( ) cujos comandos vão da linha 5 a linha 13.
  • Linhas de 6 a 10 : verifica se o número foi fornecido.
  • Linha 12 : converte número.
  • Linha 13 : exibe número.

Execução

Para compilar, digite

gcc -o atoi atoi.c
Para executar, basta digitar, por exemplo,
./atoi 4
A resposta é
Numero recebido = 4

Observações

  • Os espaços em branco no início da string são ignorados.
  • Se a string é composta por conjuntos de caracteres separados por espaços em branco, apenas o primeiro conjunto de caracteres é considerado na conversão.
  • Se a string é um conjunto vazio ou possui caracteres não numéricos, a função retorna o valor zero.
