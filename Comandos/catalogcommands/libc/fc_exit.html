exit( )

Descrição

Esta função termina o processo corrente.

Formato

#include<stdlib.h>
void exit(int status);
  • O argumento status corresponde ao código de saída do processo. Este parâmetro é retornado ao processo-pai.

Exemplo

1. #include<stdio.h>
2. #include<stdlib.h>
3. int main(int argc, char * const argv[])
4.    {
5.       int num;
6.       if (argc != 2)
7.          {
8.             printf("Forneca um numero\n");
9.             exit(-1);
10.         }
11.      else
12.         num = atoi(argv[1]);
13.      if (num < 0)
14.         {
15.            printf("numero menor que zero\n");
16.            exit(1);
17.         }
18.      else
19.         if (num <= 100)
20.            {
21.               printf("numero entre zero e 100\n");
22.               exit(2);
23.            }
24.         else
25.            {
26.               printf("numero maior que 100\n");
27.               exit(3);
28.            }
29.   }

Comentários

  • Linhas de 1 a 2 : é feita a inclusão dos arquivos headers da biblioteca C.
    • stdio.h - cabeçalho com a definição das funções de E/S dos arquivos como, por exemplo, printf( ).
    • stdlib.h - cabeçalho com a definição da função atoi( ).
  • Linha 3 : início da função main( ) cujos comandos vão da linha 5 a linha 28.
  • Linha 5 : define variável.
  • Linhas de 6 a 10 : verifica se o número foi fornecido.
  • Linha 12 : converte número usando a função atoi( ).
  • Linhas de 13 a 17 : verifica se número é menor que zero.
  • Linhas de 19 a 23 : verifica se número é menor que ou igual a cem.
  • Linhas de 24 a 28 : verifica se número é maior que cem.

Execução

Para compilar, digite

gcc -o exit exit.c
Para executar, forneça o nome do programa e um número. Por exemplo,
./exit 200
Para verificar, o valor retornado com a função exit( ), digite
echo $?
No exemplo acima, a resposta é
3

Observações

O processo que executa a função exit( ) é encerrado imediatamente. Além disso,

  • se existem arquivos ainda abertos, estes arquivos são fechados;
  • todas as funções registradas com atexit( ) e on_exit() são chamadas na ordem inversa em que foram registradas;
  • o buffer padrão de E/S é limpo.
