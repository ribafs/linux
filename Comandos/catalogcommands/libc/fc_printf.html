printf( )

Descrição

Função que formata e imprime dados.

Formato

#include<stdio.h>
int printf(const char *formato, lista)
  • Os argumentos formato e lista correspondem, respectivamente, a uma string com o formato dos dados a serem exibidos e a uma lista de dados (variáveis e/ou valores).
  • A função printf( ) imprime a cadeia de caracteres definida no argumento formato, interpretando as diretivas '%' e as teclas de escape '\'.
  • São exemplos de diretivas que podem ser usadas na função printf( ):

    FormatoSignificado
    %ahexadecimal ponto flutuante, letra minúscula
    %Ahexadecimal ponto flutuante, letra maiúscula
    %ccaractere (tipo char)
    %dvalor inteiro (tipo int)
    %enotação científica, letra minúscula
    %Enotação científica, letra maiúscula
    %fvalor com ponto flutuante (tipo float), letra minúscula
    %Fvalor com ponto flutuante (tipo float), letra maiúscula
    %gusa a menor representação (%e ou %f)
    %Gusa a menor representação (%E ou %F)
    %ivalor inteiro (tipo int)
    %ldvalor inteiro longo (tipo long int)
    %luvalor decimal longo (tipo unsigned long int)
    %ovalor octal (tipo unsigned int)
    %sstring (tipo *char)
    %uvalor decimal sem sinal (tipo unsigned int)
    %xvalor hexadecimal (tipo unsigned int)
  • São exemplos de teclas de escape na função printf( )
    TeclaSignificado
    \\caractere de barra invertida
    \'apóstrofo
    \"aspas
    \?interrogação
    \acaractere de aviso sonoro
    \bcaractere de retrocesso
    \fcaractere de avanço
    \nnova linha
    \rretorno de carro
    \ttabulação horizontal
    \vtabulação vertical

Exemplo

1. #include<stdio.h>
2. int main()
3.    {
4.        printf("Processo: PID = %ld, PPID = %ld\n", (long)getpid(), (long)getppid());
5.    }

Para obter o código-fonte do exemplo clique aqui.

Comentários

  • Linha 1 : é feita a inclusão do arquivo header da biblioteca C.
    • stdio.h - cabeçalho com a definição das funções de E/S dos arquivos como, por exemplo, printf( ).
  • Linha 2 : início da função main( ) cujos comandos vão da linha 3 a linha 5.
  • Linha 4 : exibe o PID e o PPID do processo.

Execução

Para compilar, digite

gcc -o printf printf.c
e para executar, digite
./printf

Uma possível saída para a execução do programa é mostrada abaixo.

Processo: PID = 2527, PPID = 2262

Podemos também observar que o PPID do processo-pai é, neste exemplo, 2262. Este valor correponde ao PID do shell, já que o programa é executado a partir da linha de comandos. Para obter o PID do shell, use o comando ps. Para o exemplo acima, o sistema apresenta uma resposta semelhante a

PIDTTYTIMECMD
2262pts/000:00:00bash
3370pts/000:00:00ps
onde podemos notar que o usuário usa o shell bash.
