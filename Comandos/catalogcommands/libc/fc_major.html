major( ) / minor( )

Descrição

Estas funções ajudam a gerenciar o número do device.

Formato

#include <sys/types.h>
unsigned int major(dev_t dev);
unsigned int minor(dev_t dev);
  • O argumento dev corresponde ao valor decimal da combinação do major device number com o minor device number.

Exemplo

Veja a discussão do programa lstat.c nas chamadas de sistema stat()/lstat()/fstat().

Observações

O número do device é composto de duas partes:

  1. O major device number que identifica o driver associado com o device;
  2. O minor device number que identifica o device.
A partição /dev/sda5 em um disco SCSI tem major number igual a 8 (padrão para SCI) e minor number igual a 5. O comando stat pode ser usado para verificar os valores desses campos.
