timeval

Descrição

Estrutura usada para armazenar intervalo de tempo.

Cabeçalho

Para usar esta estrutura de dados deve-se incluir o cabeçalho sys/time.h no programa.

#include <sys/time.h>

Formato

struct timeval
{
    int time_t tv_sec; /* seconds */
    suseconds_t tv_usec; /* microseconds */
};

Campos

  • tv_sec - o número de segundos do tempo contabilizado.
  • tv_usec - o resto do tempo contabilizado em microsegundos.

Observações

  • A estrutura timeval é usada pela estrutura itimerval( ).
