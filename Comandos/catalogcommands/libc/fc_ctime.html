ctime( )

Descrição

Esta função converte um tempo do Unix para um formato compreensível ao ser humano ("dia-da-semana mês dia hh:mm:ss ano\n").

Formato

#include <time.h>
char *ctime(const time_t *timep);
  • O argumento timep representa o número de segundos que existe entre a meia-noite de 01/01/1970 (tempo do Unix) e um determinado horário.

Valor de retorno

Retorna o tempo correspondente em formato humano ou NULL em caso de erro.

Exemplo

1. #include <stdio.h>
2. #include <time.h>
3. int main()
4.     {
5.        time_t tp;
6.        time(&tp);
7.        printf("%s", ctime(&tp));
8.     }

Comentários

  • Linhas de 1 a 2 : é feita a inclusão dos arquivos headers da biblioteca C.
    • stdio.h - cabeçalho com a definição das funções de E/S dos arquivos como, por exemplo, printf( ).
    • time.h - cabeçalho com as funções para manipular datas e horas.
  • Linha 3 : início da função main( ) cujos comandos vão da linha 5 a linha 7.
  • Linha 5 : define uma variável do tipo time_t que corresponde ao tipo long int (inteiro longo).
  • Linha 6 : carrega tempo atual destro da estrutura.
  • Linha 7 : exibe a data atual.

Execução

Para compilar, digite

gcc -o ctime ctime.c
Para executar, basta digitar
./ctime
