install

install [opções] arquivo(s)

Descrição

Este comando copia arquivos alterando permissões, dono e grupo dos arquivos. Ele combina os comandos cp com chown e chmod.

Algumas opções do comando

  • -b : cria um backup do arquivo destino que já existe, antes de sobrescrevê-lo.
  • --help : mostra as opções do comando.
  • -v ou --verbose : informa o que está sendo feito.
  • --version : informa a versão do aplicativo.

Exemplos

  • Para o usuário aluno copiar o arquivo /etc/passwd para o arquivo teste, basta digitar
    install /etc/passwd teste
    Abaixo são mostrados os atributos dos dois arquivos. Note que o arquivo teste tem permissão para execução.
    -rw-r--r-- 1 root root 1925 Mar 14 18:00 /etc/passwd
    -rwxr-xr-x 1 aluno aluno 1925 Mai 16 16:34 teste
  • Para copiar o arquivo /etc/passwd dentro do diretório teste (já existente), digite
    install /etc/passwd teste
    Neste caso, o arquivo passwd é criado dentro do diretório teste.
    -rwxr-xr-x 1 aluno aluno 1925 Mai 16 16:34 passwd
  • Para copiar o arquivo teste para o arquivo teste2 e manter um backup deste arquivo (o arquivo teste2 já existe), basta digitar
    install -b teste teste2
