tex

Descrição

O TeX é um sistema de tipografia bastante utilizado no meio acadêmico por ser capaz de produzir fórmulas e símbolos matemáticos de uma forma elegante. A geração de documentos com TeX pode ser dividida em quatro etapas:

  1. O uso de um editor de texto qualquer para digitar o documento com extensão .tex. O arquivo criado possui macros que definem como o texto deverá ser formatado.
  2. Geração do arquivo com extensão .dvi com o comando latex.
  3. Conversão do arquivo .dvi em um arquivo .ps (PostScript) com o comando dvips.
  4. Visualização do arquivo com extensão .ps com o comando gv.

Exemplo

Abaixo é mostrado um pequeno exemplo de um arquivo com macros tex. Digite este arquivo e salve-o com o nome de teste.tex.

\documentclass[12pt]{article}
\usepackage[brazil]{babel}
\usepackage[utf8]{inputenc}
\begin{document}
\section{Introdução}
\label{sec:introducao}
Olá, {\em Mundo}.
\end{document}
  • A primeira linha do arquivo define o tipo de documento a ser formatado: é um artigo e a letra padrão do texto é de tamanho 12.
  • As duas linhas seguintes incluem os pacotes necessários para reconhecer acentuação no texto (o padrão é o inglês e, portanto, sem acento) e definir a codificação UTF-8 (padrão DuZeru).
  • A quarta linha informa o início do documento.
  • A quinta linha indica o início de uma seção do artigo com o título "Introdução" (a numeração das seções é feita pelo utilitário).
  • A sexta linha define um label para a seção. Isto permite que a seção seja referenciada em qualquer parte do texto. No caso de alteração na numeração dessa seção, o aplicativo automaticamente renumerará todas as citações.
  • A sétima linha mostra a frase "Olá, {\em Mundo}", onde a tag \em indica que a palavra "Mundo" aparecerá em itálico.

Com o arquivo pronto, o próximo passo é gerar o arquivo teste.dvi com o comando

latex teste.tex

Para converter o arquivo dvi para o formato postscript digite

dvips -o teste.ps teste

Para visualizar o texto, basta digitar

gv teste.ps

Observações

  • Não é obrigatório o uso da extensão .tex nos arquivos com macros tex.
  • Para instalar o tex, basta digitar
    sudo apt-get install texlive-latex-base

Tópicos relacionados

  • dvips - converte arquivo .dvi em .ps (PostScript).
  • gv - aplicativo para visualização de arquivos com extensão ps e pdf.
  • latex - gera arquivo .dvi a partir da definição de macros TEX.
  • ps2pdf - converte arquivo no formato ps para o formato pdf.
