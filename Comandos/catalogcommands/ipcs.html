ipcs

ipcs [opções]

Descrição

Este comando fornece informações sobre recursos ipc (interprocess communication).

Algumas opções do comando

  • -m : informações sobre segmentos de memória compartilhada.
  • -q : informações sobre filas de mensagens.
  • -s : informações sobre semáforos.
  • -a : informações sobre todos os recursos (é o padrão).

Observações

  • O uso do comando ipcs, sem parâmetros, faz com que o sistema forneça informações sobre todos os recursos.

Tópicos relacionados

  • ipcrm - remove recurso ipc (interprocess communication).
