xargs

xargs [opções]

Descrição

Este comando constroi e executa comandos a partir da entrada padrão.

Algumas opções do comando

  • -a arquivo : usa o arquivo especificado como entrada padrão.
  • --help : exibe as opções do utilitário.
  • --version : mostra informações sobre o utilitário.

Exemplos

  • Para deletar todos os arquivos que terminam com ~ (versão anterior do arquivo) a partir do diretório linux, basta digitar
    find linux/*~ | xargs rm -fr
  • Para procurar a palavra linux em arquivos com nome teste.txt, digite
    find -name teste.txt | xargs grep linux
    Note que é possível ter mais de um arquivo com o mesmo nome, pois o comando find faz a pesquisa em todos os diretórios abaixo do diretório atual.
  • Para procurar a palavra DuZeru nos arquivos do diretório linux e gravar o resultado no arquivo teste, digite
    find linux | xargs grep -s -a -i DuZeru > teste
