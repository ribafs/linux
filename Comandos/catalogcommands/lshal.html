lshal

lshal [opções]

Descrição

  • Este comando exibe informações detalhadas sobre o hardware. Na realidade, este aplicativo analisa o diretório /proc e gera uma saída costumizada.
  • HAL (hardware abstraction layer) é o nível de abstração entre o hadware e o kernel do sistema operacional da máquina. Portanto, este software permite que um mesmo kernel seja executado em máquinas com diferentes hardwares.

Algumas opções do comando

  • -s, --short : apresenta um relatário resumido.
  • -t, --tree : o relatório apresenta o relatório organizado em uma ávore.
  • -u, --show : mostra o relatório apenas de um UDI (Unique Device Identifier).
  • -h, --help : informa as opções do aplicativo.
  • -V, --version : exibe a versão do aplicativo.
