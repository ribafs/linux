depmod

depmod [opções]

Descrição

O comando verifica as dependências dos módulos do kernel e grava o resultado no arquivo /lib/modules/versão do kernel/modules.dep. Por exemplo, para a versão 3.18.1-031801-generic do kernel, o arquivo gerado será /lib/modules/3.18.1-031801-generic/modules.dep.

Algumas opções do comando

  • -a : verifica as dependências de todos os módulos do kernel. Esta opção é usada na ausência de qualquer outra opção.
  • -b diretório : define o nome do diretório base para gravar o arquivo de saída.
  • -h : fornece informações sobre o comando.
  • -n : direciona o arquivo com as dependências para a saída padrão.
  • -v : informa os módulos processados.
  • -V : informa a versão do utilitário.

Tópicos relacionados

  • lsmod : exibe informações sobre os módulos que estão instalados no kernel.
  • modinfo - mostra informações sobre um módulo do kernel.
  • modprobe : carrega e remove módulos do kernel.
  • insmod : insere um módulo no kernel do Linux.
  • rmmod : retira um módulo do kernel do Linux.
  • uname -r : verifica a versão do kernel do Linux.
