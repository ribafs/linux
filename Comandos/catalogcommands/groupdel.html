groupdel

groupdel grupo

Descrição

Este comando deleta um grupo de usuários do sistema.

Exemplo

O comando

groupdel teste
apaga o grupo chamado teste.

Observações

  • A lista dos grupos existentes no sistema é armazenada em /etc/group.
  • Antes de usar o comando groupdel, deve-se remover os usuários do grupo. O sistema não faz qualquer tipo de verificação.
  • O comando delgroup também deleta um grupo de usuários.
  • Para criar grupos, use o comando groupadd ou o comando addgroup.
