rev

rev [arquivos]
  • O argumento arquivos correponde aos nomes dos arquivos de entrada.

Descrição

Este comando copia o arquivo especificado para a saída padrão invertendo os caracteres de cada uma das linhas.

Algumas opções do comando

  • -h, --help : exibe as opções do aplicativo.
  • -V, --version : mortra informações sobre o aplicativo.

Exemplo

Suponha que o arquivo teste possui o seguinte conteúdo:

teste1
12345
teste2
abcdefgh
9876543210
O comando
rev teste
mostra a seguinte saída
1etset
54321
2etset
hgfedcba
0123456789

Observações

  • Caso o nome de um arquivo não seja definido junto com o comando rev, o sistema usará a entrada padrão para receber os dados.

Tópicos relacionados

  • cat - concatena e/ou exibe um ou mais arquivos.
  • less : permite paginar o conteúdo de um arquivo.
  • more : permite paginar o conteúdo de um arquivo.
  • tac : concatena e/ou exibe um ou mais arquivos, onde as linhas dos arquivos aparecem na ordem decrescente (a última linha aparece primeiro e a primeira linha aparece por último).
