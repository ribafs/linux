lsscsi

lsscsi [opções]

Descrição

Este comando lista os dispositivos scsi/sata como HDs.

Algumas opções do comando

  • -l ou --long : exibe informação adicional para cada dispositivo SCSI.
  • -L ou --list : exibe informação adicional nos pares =.
  • -p ou --protection : exibe informação adicional sobre a integridade dos dados.
  • -v ou --verbose : exibe o nome dos diretórios de onde as informações foram retiradas.

Exemplos

  • Exemplo de saída padrão
    lsscsi
    [1:0:0:0] cd/dvd VBOX CD-ROM 1.0 /dev/sr0
    [2:0:0:0] disk ATA VBOX HARDDISK 1.0 /dev/sda
  • Exemplo de saída usando
    lsscsi -c
    Attached devices:
    Host: scsi1 Channel: 00 Target: 00 lun: 00
    Vendor: VBOX Model: CD-ROM REV: 1.0
    Type: CD-ROM   ANSI SCSI revision: 50
    Host: scsi2 Channel: 00 Target: 00 lun: 00
    Vendor: ATA Model: VBOX HARDDISK REV: 1.0
    Type: Direct-Access   ANSI SCSI revision: 50
  • Exemplo de saída usando
    lsscsi -l
    [1:0:0:0] cd/dvd VBOX CD-ROM 1.0 /dev/sr0
    state=running queue_depth=1 scsi_level=6 type=5 device_blocked=0 timeout=30
    [2:0:0:0] disk ATA VBOX HARDDISK 1.0 /dev/sda
    state=running queue_depth=31 scsi_level=6 type=5 device_blocked=0 timeout=30

Observações

  • Para usar lsscsi, é preciso instalá-lo, através do comando
    sudo apt-get install lsscsi
