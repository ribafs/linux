/bin

Descrição

  • Este diretório possui os comandos binários utilizados durante o processo de inicialização do sistema, quando nenhum outro sistema de arquivos ainda está montado.
  • Todos os usuários do sistema tem acesso aos aplicativos deste diretório. Entretanto, isto não significa que os usuários tenham permissão para executar qualquer coisa com estes aplicativos.

Exemplos de arquivos

  • cat - concatena arquivos e lista o resultado na saída padrão.
  • chmod - altera as permissões de acesso a arquivos do sistema.
  • cp - copia arquivos e diretórios.
  • ls - lista o conteúdo dos diretórios.
  • mkdir - cria diretórios.
  • mv - move/renomeia arquivos/diretórios.
  • rm - apaga arquivos e/ou diretórios.
