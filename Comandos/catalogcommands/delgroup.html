delgroup

delgroup [options] [grupo]

Descrição

Este comando remove um grupo do sistema.

Algumas opções do comando

  • --help : mostra as opções do aplicativo.
  • --system : especifica que o grupo a ser deletado é do sistema.
  • --version : mostra a versão do aplicativo.

Exemplo

Para deletar o grupo alunos, basta digitar:

delgroup alunos

Observações

  • O comando groupdel também deleta um grupo de usuários.
  • Para criar grupos, use o comando groupadd ou o comando addgroup.
